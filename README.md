### Search-github-docs
<h6>快速打开 github 主页：Enter</h6>
<h6>快速搜索仓库：输入仓库名 + Enter</h6>
<h6>快速搜索用户：输入 u + 空格 + 用户名 + Enter</h6>

> 由于 github 的 api 存在调用频率限制 (未验证用户每分钟最多 10 次)，所以如果出现提示 `Rate Limit`，请稍等再试即可。
